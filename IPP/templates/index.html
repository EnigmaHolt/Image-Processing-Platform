Convolutional Neural Networks

Researchers have found that a kind a of model called a deep convolutional neural network can achieve reasonable performance on hard visual recognition tasks -- matching or exceeding human performance in some domains.

TensorFlow

TensorFlow is an Open Source Software Library for Machine Intelligence.The flexible architecture allows you to deploy computation to one or more CPUs or GPUs in a desktop, server, or mobile device with a single API.

Inception-v3

Inception-v3 is the latest model released by Google for running recogintion. It is trained for the ImageNet Large Visual Recognition Challenge using the data from 2012.

CIFAR-100 Dataset

CIFAR-100 Dataset is an open source dataset which was collected by Alex Krizhevsky, Vinod Nair, and Geoffrey Hinton. It consists of 60000 32x32 color images in 100 classes, with 600 images per class.
